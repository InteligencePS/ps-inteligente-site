# PS Inteligente

Site estático para publicação no GitHub Pages.

## Publicação

1. Crie um repositório público no GitHub.
2. Envie todos os arquivos e pastas deste projeto.
3. Abra `Settings` → `Pages`.
4. Em `Source`, escolha `Deploy from a branch`.
5. Selecione `main` e `/root`.
6. Salve.

O arquivo inicial é `index.html`.

## Edição

- Aparência: `assets/style.css`
- Busca e tema: `assets/script.js`
- Página inicial: `index.html`
- Conteúdos: pastas `protocolos`, `prescricoes`, `escores` e `modelos`

## Segurança

O conteúdo clínico incluído é demonstrativo e deve ser revisado antes do uso assistencial.
